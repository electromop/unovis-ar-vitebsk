# УНОВИС × Витебск — AR-плакат

Интерактивный AR-проект: плакат в духе супрематизма 1920-х + камера телефона = оживающие геометрические формы. Идея — переосмысление УНОВИСа (Малевич, Лисицкий и ко., Витебск 1919–1922) в 2026 году.

## Что внутри

```
unovis-ar-vitebsk/
├── index.html      ← лендинг с QR-кодом и описанием проекта
├── ar.html         ← сама AR-сцена (MindAR + A-Frame)
├── poster.svg      ← плакат в стиле УНОВИС, он же маркер
├── poster.png      ← тот же плакат в PNG (нужен для компиляции маркера)
├── targets.mind    ← СГЕНЕРИРОВАТЬ (см. шаг 2)
└── README.md
```

## Стек

- **A-Frame 1.5** — WebXR-движок (CDN)
- **MindAR.js 1.2.5** — image tracking в браузере (CDN)
- **SVG + ванильный JS** — плакат и лендинг
- Никаких приложений, сборщиков, npm. Просто HTML.

## Запуск (5 шагов)

### 1. Подготовь маркер (PNG из SVG)

Уже готов — `poster.png`. Если что-то правил в `poster.svg`, пересоздай PNG:

```bash
# на любой машине, где стоит ImageMagick или Inkscape
rsvg-convert -w 800 poster.svg > poster.png
# или
inkscape poster.svg --export-type=png --export-filename=poster.png -w 800
```

### 2. Скомпилируй `.mind`-файл

`.mind` — это «отпечаток» плаката, который MindAR использует для распознавания.

1. Открой в браузере: **https://hiukim.github.io/mind-ar-js-doc/tools/compile**
2. Выбери «Image Target Compiler»
3. Загрузи `poster.png`
4. Нажми **Start** → подожди
5. Нажми **Download** → получишь `targets.mind`
6. Положи его в корень проекта рядом с `ar.html`

### 3. Локальный запуск (для проверки)

MindAR требует HTTPS (или localhost) — файл `file://` не подойдёт.

```bash
# python (проще всего)
cd unovis-ar-vitebsk
python3 -m http.server 8080

# или node
npx serve .
```

Открой с телефона в той же Wi-Fi-сети: `http://<твой-локальный-ip>:8080/ar.html`

⚠️ На iOS Safari требует HTTPS даже для локалки — используй **ngrok** или сразу деплой на GitHub Pages (шаг 4).

### 4. Деплой на GitHub Pages (бесплатно, 2 минуты)

```bash
# в папке unovis-ar-vitebsk
git init
git add .
git commit -m "UNOVIS x Vitebsk AR"
git branch -M main
git remote add origin https://github.com/<твой-ник>/unovis-ar-vitebsk.git
git push -u origin main
```

Затем в GitHub: **Settings → Pages → Branch: main → /root → Save**.

Через 1–2 минуты получишь ссылку вида `https://<ник>.github.io/unovis-ar-vitebsk/`. Она уже по HTTPS — всё заработает.

Альтернативы: **Netlify Drop** (просто перетащи папку на netlify.com/drop), **Vercel**, **Surge.sh**.

### 5. Напечатай плакат и проверь

1. Открой `poster.svg`, распечатай на A4/A3 (лазерник/любая типография)
2. На телефоне зайди на `https://<ник>.github.io/unovis-ar-vitebsk/`
3. Нажми «ЗАПУСТИТЬ AR», разреши камеру
4. Наведи на напечатанный плакат
5. 🎨 Плакат оживает: парящий красный квадрат, вращающийся чёрный круг, появляющаяся надпись УНОВИС

## Что показывает AR

Когда камера видит плакат, над ним появляются:

- **Красный квадрат** (омаж Малевичу) — вращается и парит
- **Чёрный круг с белым кольцом** — плавает по горизонтали
- **Синий треугольник** — медленно вращается
- **Жёлтая полоса** — пульсирует
- **Красная диагональ, чёрные бруски, красный круг** — появляются по очереди
- **Крест-архитектон** (3D-объект из 3 брусков) — крутится
- **Надпись «УНОВИС · VITEBSK 1919–1922»** — проявляется над плакатом

Всё анимировано через A-Frame animation component — легко редактируется, см. `ar.html`, блок `<a-entity mindar-image-target>`.

## Как дорабатывать

- **Другой плакат** → новый PNG → новый `.mind` → меняй `imageTargetSrc`
- **Несколько плакатов** (серия по Витебску) → скомпилируй их в один `.mind` разом, затем делай `<a-entity mindar-image-target="targetIndex: 0">`, `targetIndex: 1`, и т.д.
- **3D-модель** вместо примитивов → добавь `<a-assets>` с `<a-asset-item id="m" src="model.glb">` и `<a-gltf-model src="#m">`
- **Звук при обнаружении** → `<a-sound src="url(hit.mp3)" autoplay="false" on="targetFound">`
- **Маршрут по городу** — сделай карту с точками, на каждой — свой плакат и свой AR-слой

1. **Сохрани процесс** — скриншоты компиляции, печати, тестов. В дипломе хорошо смотрится «как делалось»

3. **Параллель с оригинальным УНОВИСом** — они расписывали трамваи, мы «расписываем» городское пространство цифровым слоем
4. **Видео-демка** — сними на телефон, как плакат оживает. Это будет главный «wow» при защите
5. **QR-коды** — распечатай их рядом с плакатами, чтобы прохожие могли запустить AR без набора URL

## Проблемы и решения

| Проблема | Решение |
|---|---|
| Камера не запускается | Проверь HTTPS, разреши доступ к камере в настройках браузера |
| Плакат не распознаётся | Напечатай крупнее (A3 лучше), хорошее освещение, без бликов |
| iOS: «не поддерживается» | Обнови iOS до 15+, используй Safari (не Chrome на iOS) |
| Долго загружается | Нормально при первом заходе — скачивает MindAR (~1Мб) |
| Всё дёргается | Ракурс: держи камеру параллельно плакату, ~30–50 см |
